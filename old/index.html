Position

Simple position declarations for absolute, relative, fixed and sticky.

.left( @top) or .x( @left )

        /* less */
        .foo{
          #relative > .x(20px);
        }
        

        /* css output */
        .foo{
          position: relative;
          left: 20px;
        }

Type

Shorthands for working with type.


        /* font-weight & font-style */
        .bold{
          #type > .bold();
        }

        .light{
          #type > .light();
        }

        .italic{
          #type > .italic();
        }

        .oblique{
          #type > .italic();
        }

        /* text-transform */
        .uppercase{
          #type > .uppercase();
        }

        .lowercase{
          #type > .lowercase();
        }

        .capitalize{
          #type > .capitalize();
        }

        .underline{
          #type > .underline();
        }

        .overline{
          #type > .overline();
        }

        /* font-variant */
        .smallcaps{
          #type > .smallcaps();
        }

CSS3

        .one{

        .two{

          #css3 > .border-radius(5px,10px,0px,40px);

Reset

Reset common .css properties to their default values.


        /* less */
        .foo{
          padding: 20px;
          text-transform: uppercase;
          border: 1px solid #0EBFE9;
          &:hover{
            #reset > ._(padding);
            #reset > ._(text-transform);
            #reset > ._(border);
          }
        }
hi
The #reset > ._(@property) mixin will accept any of the following properties
