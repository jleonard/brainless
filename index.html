Position

Simple position declarations for absolute, relative, fixed and sticky.
Note hi there.
.left( @top) or .x( @left )

        /* less */
        .foo{
          #relative > .x(20px);
        }
        

        /* css output */
        .foo{
          position: relative;
          left: 20px;
        }
