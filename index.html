Examples

1. typography control with text.less

  
    p{
      #text > .bold();
      #text > .uppercase();
      #text > .underline();
    }
  
  
    /* css output */
    p{
      font-weight: bold;
      text-transform: uppercase:
      text-decoration: underline;
    }

2. element positioning with position.less

Use the #absolute, #relative and #fixed groups to control element positioning.
  
    /* less */
    div{
      #absolute > .bottom-right(10px,15px);
    }
  
  
    /* css output */
    div{
      position: absolute;
      bottom: 10px;
      right: 15px;
    }

3. css transform management with transform.less

  
    /* 
     * less
     * The mixins allow you to chain your transforms together. 
     */
    div{
      #transform > .rotateX(45deg);
      #transform > .skew(30deg,25deg);
    }
  
  
    /* css output */
    div{
      -webkit-transform : rotateX(45deg) skew(30deg,25deg);
      -moz-transform    : rotateX(45deg) skew(30deg,25deg);
      -o-transform      : rotateX(45deg) skew(30deg,25deg);
      -ms-transform     : rotateX(45deg) skew(30deg,25deg);
      transform         : rotateX(45deg) skew(30deg,25deg);
    }

4. centering with center.less

  
    div{
      position: absolute;
      #center > .y();
    }
  
  
    /* css output */
    div{
      position          : absolute;
      top               : 50%;
      -webkit-transform : translateY(-50%);
      -moz-transform    : translateY(-50%);
      -o-transform      : translateY(-50%);
      -ms-transform     : translateY(-50%);
      transform         : translateY(-50%);
    }

5. css keyframe animation control with animation.less

  
    div{
      #animation > .name(wiggle);
      #animation > .duration(0.25s);
      #animation > .loop();
    }
  
  
    @keyframes wiggle {
      0% {transform:rotate(1deg);}
      100% {transform:rotate(-1deg);}
    }
    /* css output */
    div{
      animation-name: wiggle;
      -webkit-animation-name: wiggle;

      animation-duration: 0.25s;
      -webkit-duration: 0.25s;

      animation-direction: alternate;
      -webkit-animation-direction: alternate;

      animation-iteration-count: infinite;
      -webkit-iteration-count: infinite;
    }
