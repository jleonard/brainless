brain.less

A .less framework to simplify .css development.

Examples

1. typography control with text.less

  
    p{
      #text > .bold();
      #text > .uppercase();
      #text > .underline();
    }
  
  
    /* css output */
    p{
      font-weight: bold;
      text-transform: uppercase:
      text-decoration: underline;
    }

2. element positioning with position.less

Use the #absolute, #relative and #fixed groups to control element positioning.
  
    /* less */
    div{
      #absolute > .top-left(25px,50px);
    }
  
  
    /* css output */
    div{
      position: absolute;
      top: 25px;
      left: 50px;
    }

3. css transform management with transform.less

Example

Code

  
    /* 
     * less
     * The mixins allow you to chain your transforms together. 
     */
    div{
      #transform > .scaleX(25%);
      #transform > .rotateY(50%);
    }
  
  
    /* css output */
    div{
      -webkit-transform : scaleX(25%) rotateY(50%);
      -moz-transform    : scaleX(25%) rotateY(50%);
      -o-transform      : scaleX(25%) rotateY(50%);
      -ms-transform     : scaleX(25%) rotateY(50%);
      transform         : scaleX(25%) rotateY(50%);
    }

4. centering with center.less

  
    div{
      #center > .x();
    }
  
  
    /* css output */
    div{
      position          : absolute;
      left              : 50%;
      -webkit-transform : translateX(-50%);
      -moz-transform    : translateX(-50%);
      -o-transform      : translateX(-50%);
      -ms-transform     : translateX(-50%);
      transform         : translateX(-50%);
    }

5. css keyframe animation control with animation.less

  
    div{
      #animation > .name(rotate);
      #animation > .duration(4s);
      #animation > .loop();
    }
  
  
    /* css output */
    div{
      animation-name: rotate;
      -webkit-animation-name: rotate;

      animation-duration: 4s;
      -webkit-duration: 4s;

      animation-direction: alternate;
      -webkit-animation-direction: alternate;

      animation-iteration-count: infinite;
      -webkit-iteration-count: infinite;
    }
